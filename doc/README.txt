RUNNING

    The tool can be run from the command line as an executable jar file, as:

        java -jar relaxws.jar [-d output-dir] [-encoding source-encoding] <intput-file>

BUILDING

    Requires ant 1.7, jdk 1.5, and javacc 5.0 to build.  See build.xml for instructions.
